# 设置Kilo Code

注册Kilo Code时，你将获得**价值<FreeTierAmount />的免费token**作为启动资源。领取免费token的步骤：

1. **注册：** 完成注册流程
2. **首次充值：** 向账户添加资金（最低 $5）并获得 $20 奖励积分
3. **开始编程：** 尽情使用你的 $20 免费积分

:::note 关于临时冻结
有时，我们可能需要临时冻结 $0.50，这仅用于身份验证：

- **免费** - 这是临时授权，不会被收取费用
- **即时释放** - Kilo Code 在验证后会立即释放冻结
- **银行处理** - 虽然我们即时释放，但资金返回可能需要几个工作日（因银行而异）
  :::

## 注册流程

Kilo Code提供简单的注册流程，通过你的账号即可使用前沿的编程模型：

1. 在扩展中点击"免费试用 Kilo Code"
1. 使用你的 Google 账户登录 kilocode.ai
1. kilocode.ai 将提示你打开 Visual Studio Code
   - 当使用浏览器中的 IDE 时，系统会要求你手动复制 API 密钥
1. 允许打开 VS Code 后，还必须允许 VS Code 打开授权 URL
